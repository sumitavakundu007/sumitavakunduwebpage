'rsync' documentation

rsync command is used to sync the files and folders to a directory. It won't copy the already copied files and folders

'--max_size' flag will help you to copy the files less than a particular size

'--exclude' flag will help you to exclude some particular directories

$ rsync -av --progress --max-size=1g --exclude={'dir3/*', 'file1.txt', 'dir1'} /src username@ip_address:/dest_path

Go to your external hard drive path where you want to keep the folders. and rsync from there

$ rsync -av --progress --max-size=1g --exclude={'dir3/*', 'file1.txt', 'dir1'} username@ip_address:/src_path .
