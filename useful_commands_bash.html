Useful bash commands

'sed' commands

Let's say you have an input file containing the following line along with other lines

"move_size_translation": 0.1

You want to change a particular parameter, say 0.1 to 0.5

You can create a file called test.sh

Open the file and add the following commands

$ #!/bin/sh
$ sed -e "s/\"move_size_translation\" : 0.1/\"move_size_translation\" : 0.5/g" <input.txt >output.txt

Make the .bash file readable

$ chmod a+x test.sh

Run the .bash file

$ ./test.sh
